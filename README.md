### Features
+ **Responsive**: Fully responsive to devices of all sizes built using TailwindCSS's utility-first classes.
+ **Forms**: Data is stored in a database and can be interacted with
+ **Carousel**: The carousel periodically shifts to a new image to showcase all available dishes built using react-responsive-carousel
